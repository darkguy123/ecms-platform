# **App Name**: JusticeFlow CMS

## Core Features:

- Responsive Public Website: A fully responsive public website featuring dynamic pages like Landing, About, Features/Modules, Solutions, Security, Partners, and Contact. Content for these pages will be fetched from Firestore, ensuring a clean, modern, and professional UI across all devices.
- Secure Admin Panel Authentication: Protected admin dashboard access secured with Firebase Authentication (Email + Password) for login/logout functionality.
- Dynamic Page Content Management: Admin users can edit homepage text, update sections on various public pages (About, Solutions, Security), manage contact information, and control general website content via a user-friendly interface powered by Firestore.
- ECMS Module Editor: Dedicated section in the admin panel to add, edit, and delete details for each ECMS module (e.g., eFiling, eCase Management), including title, description, and key features, stored in Firestore.
- Contact Form & Message Management: Allows public website visitors to submit inquiries through a contact form, with submissions stored in Firestore. Admins can view submitted messages and mark them as read/responded.
- Image and Logo Asset Management: Admins can upload and manage images and the company logo for the website through the admin panel, with assets securely stored in Firebase Storage.
- AI-Powered Blog Post Draft Assistant: Within the admin panel's Blog/News section, an AI tool will assist administrators in drafting outlines, content ideas, or even short summaries for news articles related to court updates and product announcements, using generative AI to streamline content creation.

## Style Guidelines:

- Primary color (Corporate Green): A deep, professional green, for headings and prominent elements. RGB: #0B6B3A.
- Background color: A dark, desaturated green-gray, providing a sophisticated backdrop to elements in the dark theme. RGB: #1A2F25.
- Accent color: A vibrant lime green, used for CTAs, highlights, and interactive elements to create visual pop and draw attention. RGB: #6DF26D.
- Headline font: 'Poppins' (sans-serif), for its precise, contemporary, and avant-garde look, suitable for headlines and short impactful text.
- Body font: 'Inter' (sans-serif), chosen for its modern, objective, and neutral appearance, ensuring readability for longer blocks of text.
- Use clean, minimalistic line icons that align with a professional, government, and legal context. Icons should be clear and easily understood to enhance usability.
- Employ a modular, card-based layout with generous clean spacing to ensure content readability and a professional appearance. The design must be mobile-first and fully responsive across all devices (mobile, tablet, desktop) and include a sticky navigation bar and comprehensive footer.
- Integrate subtle, smooth animations for transitions and interactive elements to enhance user experience, such with elements loading progressively. Implement loading skeletons for improved perceived performance.
